# Studying-Phylogenetics-and-Evolution-in-Legionella---Galaxy
This repository is a companion to the chapter "Studying Phylogenetics and Evolution in Legionella" in the Methods in Molecular Biology series (Springer Nature) on "Legionella: Methods and Protocols". 

It includes Galaxy workflows and reference data. The workflows can be imported into your preferred Galaxy server by pressing the "import" button in the top right corner.

The reference data, including 16S and mip reference sequences, L. pneumophila and L. spp. assemblies, chewBBACA sampling schemes, can be downloaded [here](https://zenodo.org/records/19570332?token=eyJhbGciOiJIUzUxMiJ9.eyJpZCI6ImVkOWM0ZDk3LTZmZGQtNDc0OS05Y2FkLWIwOTkwYmIyOGUyNSIsImRhdGEiOnt9LCJyYW5kb20iOiIxY2NlZjEwYTU1YjcxZmJhNDBiODdhNTNiYzA0MGQ5ZCJ9.RUSk5aP1XJVrA7gWQmeoll5IWu9P04bY8hgMmyeIcMliUEK2WD-G0SmmVqkRqxi9QWiL_f0TAh-LiTo4gRxTHg).
